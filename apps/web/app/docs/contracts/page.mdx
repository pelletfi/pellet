## Deployer

```text
0x2cbd7730994D3Ee1aAc4B1d0F409b1b62d7C1834
```

Retired. This wallet signed the original deploy transactions and registered agent 1, but retains **zero special authority** over the contracts themselves — all three registries have no `owner`, no `admin`, no upgrade path. Anyone is free to call any public function; the deployer address is no more privileged than any other EOA.

## Compiler settings

Reproduce the exact deployed bytecode with:

```toml
# packages/hl-contracts/foundry.toml
[profile.default]
solc_version     = "0.8.28"
optimizer        = true
optimizer_runs   = 200
via_ir           = false
```

## Deployment manifest

SDKs, indexers, and downstream consumers should read addresses from that manifest rather than hard-coding.
