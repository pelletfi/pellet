Pellet ships an MCP (Model Context Protocol) server as `@pelletfi/mcp` on npm. Install it once, and your agent gets tools for every Pellet endpoint — list stables, inspect peg, fetch flows, request editorial briefings.

Every value the MCP server exposes is an **Open-Ledger Interface (OLI)** measurement — read directly from Tempo's public ledger, never aggregated from off-chain oracles. An agent calling Pellet's MCP is practicing OLI on your behalf. See [What is OLI?](/docs/oli).

## Install

```bash
npm install -g @pelletfi/mcp
```

## Configure

Add to your Claude Desktop config (`~/Library/Application Support/Claude/claude_desktop_config.json` on macOS):

```json
{
  "mcpServers": {
    "pellet": {
      "command": "npx",
      "args": ["@pelletfi/mcp"],
      "env": {
        "PELLET_KEY": "pk_live_...",
        "EVM_PRIVATE_KEY": "0x..."
      }
    }
  }
}
```

Two auth modes, pick one:

- **`PELLET_KEY`** — Pellet Pro subscription key. Unlimited calls, no per-request cost.
- **`EVM_PRIVATE_KEY`** — a Tempo-compatible private key holding pathUSD. Paid endpoints are settled via MPP automatically.

## Tools exposed

| Tool | Endpoint | Cost |
| --- | --- | --- |
| `list_stablecoins` | `GET /v1/stablecoins` | Free |
| `simulate_transfer` | `GET /v1/tip403/simulate` | Free |
| `get_peg` | `GET /v1/stablecoins/{addr}/peg` | $0.010 |
| `get_flows` | `GET /v1/stablecoins/flows` | $0.010 |
| `get_address` | `GET /v1/addresses/{addr}` | $0.010 |
| `get_flow_anomalies` | `GET /v1/stablecoins/flow-anomalies` | $0.020 |
| `get_reserves` | `GET /v1/stablecoins/{addr}/reserves` | $0.020 |
| `get_risk_score` | `GET /v1/stablecoins/{addr}/risk` | $0.050 |
| `get_rewards` | `GET /v1/stablecoins/{addr}/rewards` | $0.100 |
| `analyze_token` | `GET /v1/tokens/{addr}/briefing` | $0.200 |

## Example session

Once installed, ask Claude anything natural:

> "Which Tempo stable has the widest peg spread right now?"

Claude calls `list_stablecoins` (free), compares `peg.spread_bps` across the returned set, and responds. No extra glue code.

> "Analyze frxUSD for me."

Claude calls `analyze_token`. If you're on Pro, the call is free. If you're paying per request, $0.200 USDC.e is debited from your wallet and Claude returns the full briefing — 8 on-chain aggregators plus the coverage & provenance ledger.
